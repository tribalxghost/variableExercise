What is the difference between var and let?
var can be redeclared using var but let cannot be redeclared
var hoist to the top of the scope

What is the difference between var and const?
var can be redeclared and reassigned but const cannot
const does not get hoisted to the top of the page unlike var

What is the difference between let and const?
let can be reassigned but const cannot

What is hoisting?
runs var and function declaration before any other code
